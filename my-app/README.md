# Flashcards
Personal Web Application to store and revise language flashcards. 

An exercise in using the knowledge gained in fullstackopens course parts 0-7.

### Technologies used:
- Backend:
    - Node.js
    - Express.js
    - MongoDB
    - Mongoose
    - bcrypt
    - jsonwebtoken
- Frontend:
    - React
    - React Router
    - Redux w/Toolkit
    - Chakra UI
    - Dicebear API (Avatar Creator)
- Cloud / Server: 
    - AWS
    - EC2
    - S3
    - Nginx
